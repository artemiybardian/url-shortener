# URL Shortener

Microservice-based URL shortener.

## Services

| Service | Description |
|---------|-------------|
| **auth-service** | Registration, authentication, JWT |
| **shortener-service** | Short link generation, custom aliases |
| **redirect-service** | Fast HTTP redirect (high-load optimized) |
| **analytics-service** | Click stats, geo, referrers |

## Features

- Custom short links
- Click analytics
- Geo tracking
- QR code generation
